
## 冻结的集合

方法是：

    >>> f_set = frozenset("shucun")     #看这个名字就知道了
    >>> f_set
    frozenset({'h', 'u', 'c', 'n', 's'})
    >>> f_set.add("python")             #报错
    Traceback (most recent call last):
      File "<stdin>", line 1, in <module>
    AttributeError: 'frozenset' object has no attribute 'add'
    
    >>> a_set = set("github")           #这是一个可以原处修改的set
    >>> a_set
    set(['b', 'g', 'i', 'h', 'u', 't'])
    >>> a_set.add("python")
    >>> a_set
    set(['b', 'g', 'i', 'h', 'python', 'u', 't'])


### 元素与集合的关系

元素是否属于某个集合。

    >>> aset = set("python")
    >>> aset 
    set(['h', 'o', 'n', 'p', 't', 'y'])
    >>> "a" in aset
    False
    >>> "h" in aset
    True
